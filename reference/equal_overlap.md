# Compute overlap of equal values between two tables

Compute overlap of equal values between two tables

## Usage

``` r
equal_overlap(reference, target)
```

## Arguments

- reference:

  A table representing the reference state to compare against

- target:

  Another table that may overlap in values with reference

## Value

A percentage of equal values that overlap (number overlapping divided by
total elements in reference)

## Examples

``` r
if (FALSE) { # \dontrun{
equal_overlap()
} # }
```
